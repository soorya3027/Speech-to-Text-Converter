# Speech-to-Text-Converter

A Python project that converts speech from an audio file into text using the SpeechRecognition library and Google Speech API.

## Features
- Converts speech to text
- Supports WAV audio files
- Simple and beginner-friendly

## Technologies Used
- Python
- SpeechRecognition

## Files
- speech_to_text.py
- audio.wav

## Example Output

Transcription:
hi good morning Surya
